# Project: D2R Terror Zone Desktop Tracker (Python)

## 1. 개요
- 목적: 디아블로2 레저렉션의 현재 및 다음 공포 구역 정보를 화면에 표시하는 윈도우 데스크톱 앱 개발.
- 언어: Python 3.10+
- 프레임워크: Tkinter (가볍고 별도 라이브러리 최소화) 또는 PySide6/PyQt6
- 목표 형태: 단일 가상 환경에서 빌드된 .exe 실행 파일

## 2. 주요 기능 상세 요구사항
### A. 데이터 연동 (API)
- 소스: https://www.d2tz.info/api 또는 관련 공포구역 API 연동.
- 기능: API 토큰을 입력받아 `config.json` 또는 `settings.ini`에 암호화/저장하는 기능.
- 갱신: 매 분마다 데이터를 확인하여 남은 시간을 카운트다운하고, 정각에 정보를 자동 갱신.

### B. UI/UX 구성 (오버레이 스타일)
- **표시 정보**: 
  - 현재 공포 구역 (지역명 한글/영문 선택 가능)
  - 다음 예정 공포 구역 (API 지원 시)
  - 다음 갱신까지 남은 시간 (MM:SS)
- **제어 기능**:
  - **투명도 조절**: 슬라이더를 통해 배경 및 텍스트의 투명도(Alpha) 조정.
  - **Always on Top**: 항상 다른 창 위에 표시되는 기능.
  - **위치 고정 (Lock Position)**: 체크 시 마우스 드래그로 창 이동 불가.
  - **프레임리스**: 기본 윈도우 타이틀바를 제거하고 깔끔한 오버레이 형태로 구현.

### C. 설정 및 저장
- 프로그램을 처음 실행하거나 토큰이 없을 경우 토큰 입력창 팝업.
- 사용자가 설정한 위치, 투명도, 항상 위 설정값을 종료 시 저장하고 재시작 시 복구.

## 3. 기술적 제약 사항
- **라이브러리**: `requests` (API 통신), `tkextra` 또는 기본 `tkinter`, `pyinstaller` (빌드용).
- **실행 파일화**: `PyInstaller`를 사용하여 `--onefile`, `--noconsole` 옵션으로 빌드 가능한 구조로 코드 작성.
- **예외 처리**: API 호출 실패, 네트워크 연결 끊김, 잘못된 토큰 입력 시 사용자에게 알림 표시.

## 4. 최종 출력물 요구사항
- 전체 파이썬 소스 코드 (단일 파일 또는 구조화된 파일).
- API 토큰 발급 및 사용법에 대한 안내 주석 포함.
- `requirements.txt` 내용 포함.
- `PyInstaller` 빌드 커맨드 라인 제공.
